# General Utilities

Shared helper functions used across the CLI and core engine.

- **`cliHelpers.mjs`**: Utilities for filesystem operations, project validation, and running Docusaurus.
- **`configUtils.mjs`**: Pure logic for configuration merging, variable resolution, and path calculation.
- **`logger.mjs`**: Standardized logging with support for different levels (info, success, warn, error, debug).
- **`packageManager.mjs`**: Detects the available package manager (npm, yarn, pnpm, bun) to ensure cross-platform compatibility.
