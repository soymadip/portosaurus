# Project Roadmap

<br>

### Core Infrastructure

- [x] YAML-first configuration system.
- [x] Recursive variable resolution engine.
- [x] **Monorepo Restructure**: Modular package system with Bun Workspaces and Turborepo.
- [x] **Interaction Engine**: State-aware CLI prompt system with back-navigation.
- [ ] **TypeScript Conversion**: Port all packages to TypeScript for enhanced type safety and DX.

### Content & Logic

- [x] Smart configuration validation.
- [x] **API Documentation**: Comprehensive documentation of internal packages.
- [ ] **Remark Plugin**: Enable dynamic variables inside Markdown/MDX content.
- [ ] **Variable Escaping**: Fix dynamic variables being resolved inside code blocks.

### Theme & Features

- [x] Hybrid ESM architecture.
- [x] **Shelf Customization**: Options to set background color for icon canvases.
- [ ] **Mobile Optimization**: Fix popup and dock link behavior on mobile devices.
- [ ] **Docusaurus Admonitions**: Custom admonitions for notes and tasks.
- [ ] **StaticShort Integration**: Support for shortlinks in the tools section.
- [ ] **Enhanced Previews**: Support for custom positioning and zoom levels in preview components.
